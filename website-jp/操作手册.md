## 第一步：启动网站（每次开机后需做一次）

打开电脑上的"终端"（Terminal）应用，按顺序输入以下命令：

**启动后端（数据库服务）：**
```
cd ~/website-zhanshi/website-jp/backend
npm run dev
```
> 看到 `服务器运行在 development 模式下，端口 5005` 说明成功 ✓  
> **这个窗口不要关闭**

**再开一个新的终端窗口，启动前端（网页）：**
```
cd ~/website-zhanshi/website-jp/frontend
npm run dev
```
> 看到 `ready - started server on 0.0.0.0:3000` 说明成功 ✓  
> **这个窗口也不要关闭**

**打开浏览器，访问：**
- 网站前台（客户看到的）：`http://localhost:3000`
- 管理后台（你管理用的）：`http://localhost:3000/admin`

## 第二步：登录管理后台

1. 打开浏览器，访问 `http://localhost:3000/login`
2. 输入邮箱：`admin@kashiwagi-design.com`
3. 输入密码：`Admin123456`

## 第三步：添加新商品

1. 进入管理后台，点击左侧菜单"**产品管理**"
2. 点击右上角蓝色按钮"**+ 新增产品**"
3. 按照表单填写信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 产品名称 ⭐ | 必填，商品的名字 | 樱花垂坠耳环 |
| 简短描述 ⭐ | 必填，一句话介绍 | 925银手工雕刻，轻盈垂坠 |
| 详细描述 | 选填，详细故事与工艺 | （可以写很长） |
| 品类 ⭐ | 必选，从下拉菜单选 | 耳环 / 戒指 / 项链… |
| 原价（¥）⭐ | 必填，填数字 | 1280 |
| 折扣价（¥）| 选填，比原价低才有效 | 980 |
| 库存数量 | 填有多少件 | 10 |
| 材质 | 选填 | 925纯银 |
| 制作工艺 | 选填 | 手工雕刻 |
| 尺寸 | 选填 | 长3.5cm |
| 重量 | 选填 | 4g |
| 标签 | 用逗号分隔 | 樱花, 垂坠, 日式 |
| 设为精选 | 开关打开后会在首页展示 | ✓ |

4. 填完后点击底部"**创建产品**"按钮
5. 看到"产品创建成功！"提示即完成 ✓

> ⚠️ 注意：目前图片功能暂时使用系统默认占位图，如需上传真实图片，请联系技术人员配置图片上传服务。

## 第四步：修改已有商品

1. 进入"**产品管理**"页面
2. 找到要修改的商品，点击右侧"**编辑**"按钮

## 第五步：删除商品

1. 进入"**产品管理**"页面
2. 找到要删除的商品，点击右侧"**删除**"按钮
3. 弹出确认框，点击"**确认删除**"

> ⚠️ 删除后无法恢复，请谨慎操作！

## 常见问题

**Q：网站打不开怎么办？**  
A：检查终端窗口是否都还开着，如果关掉了需要重新执行第一步启动命令。

**Q：登录后提示"无效的凭据"怎么办？**  
A：检查邮箱和密码是否正确。密码区分大小写。

**Q：添加商品后前台没有显示？**  
A：刷新前台页面（按 F5 或 Ctrl+R）。如果是精选商品不显示在首页，请确认"设为精选"开关已打开。

**Q：库存显示红色数字是什么意思？**  
A：红色表示库存为 0（售罄），橙色表示库存少于 5 件，需要及时补充。

**Q：需要第一次初始化数据库怎么办？**  
在终端执行：
```
cd ~/website-zhanshi/website-jp/backend
npm run seed
```
这会自动添加样本产品和管理员账号（原有数据会被清空，谨慎使用）。
